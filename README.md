# Curso de Power BI: Do Zero ao Dashboard

Este repositório contém todo o material, bases de dados e projetos práticos desenvolvidos durante o curso de Power BI. O objetivo é transformar dados brutos em insights estratégicos através de dashboards interativos e storytelling.Sobre o Curso

## O curso aborda o fluxo completo de Business Intelligence dentro do ecossistema Microsoft:

    - ETL (Extract, Transform, Load): Limpeza e tratamento de dados com Power Query.

    - Modelagem de Dados: Criação de relacionamentos e esquemas (Star Schema/Snowflake).

    - DAX (Data Analysis Expressions): Criação de medidas, colunas calculadas e inteligência de tempo.

    - Visualização: Design de dashboards, escolha de gráficos e acessibilidade.

    - Tecnologias e Ferramentas

    - Power BI Desktop

    - Linguagem DAX

    - Power Query (M Language)

    - Excel / CSV 
    
## Estrutura do Repositório

    - /Bases_de_Dados: Arquivos .csv ou .xlsx usados nos exercícios.

    - /Dashboards: Arquivos .pbix com os projetos finalizados.

    - /Materiais_Apoio: Apostila.

    - /Imagens: Screenshots dos dashboards criados.
